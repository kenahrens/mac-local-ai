# AGENTS.md

Copy this into the root of a project as `AGENTS.md`. Cursor and Zed read it natively; Continue reads it in VS Code / VSCodium. One instructions file, every editor.

## Project

Short description of what this repo is and how it's run.

## Conventions

- Language/framework and the version.
- How to run, test, and build.
- Style notes that aren't obvious from the code.

## Don't

- Things the agent should not touch (generated files, vendored code, secrets).
